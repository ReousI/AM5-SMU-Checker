# AM5-SMU-Checker

The AM5 SMU Checker is a tool to check Bios/UEFI files before intallation.


<b>It lists the following points:</b>
- Board name and manufacturer
- UEFI version
- UEFI build date
- AGESA version
- Chipset firmware
- SMU version for AM5 CPU families

---

<b>How to use:</b>
- Drag and drop a BIOS file onto the .exe file (Only for AM5 UEFI files)
